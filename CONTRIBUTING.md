# Contributing to create-rustchain-agent

Thank you for your interest in contributing to `create-rustchain-agent`! This project is part of the RustChain ecosystem, a DePIN blockchain for vintage and retro hardware.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/create-rustchain-agent.git
   cd create-rustchain-agent
   ```
3. **Create a branch** for your changes:
   ```bash
   git checkout -b feature/my-feature
   ```

## Development Setup

- Ensure you have Python 3.8+ installed.
- Install dependencies as specified in the project README.
- Run any existing tests before submitting changes.

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or request features.
- Provide clear reproduction steps for bugs.
- Include your environment details (OS, Python version).

### Submitting Changes

1. Make your changes in your feature branch.
2. Write clear, concise commit messages.
3. Ensure your code follows existing style conventions.
4. Update documentation if your change affects user-facing behavior.
5. Push your branch and open a Pull Request.

### Pull Request Guidelines

- **Title**: Use a clear, descriptive title (e.g., `fix: correct agent config loading`).
- **Description**: Explain what changed and why.
- **Link issues**: Reference related issues with `Fixes #123` or `Closes #123`.
- **Keep it focused**: One logical change per PR.

## Code Style

- Follow PEP 8 for Python code.
- Use meaningful variable and function names.
- Add docstrings to public functions and classes.
- Keep functions small and focused on a single responsibility.

## Testing

- Add tests for new functionality when possible.
- Ensure existing tests pass before submitting.
- Test on your local environment before pushing.

## Community

- Be respectful and constructive in all interactions.
- Ask questions in issues or discussions — no question is too small.
- Join the RustChain community on [Discord](https://rustchain.org/discord) (if available).

## Bounties

Contributions to this project may qualify for RTC bounties! Check the [RustChain Bounty Board](https://github.com/Scottcjn/rustchain-bounties/issues) for active tasks and rewards.
